# SpargatAI Chatbot

## Overview
SpargatAI is a support assistant for Blockman Launcher, designed to provide help while adhering to specific guidelines.

## Requirements
- Python 3.x
- Install dependencies using `pip install -r requirements.txt`

## Usage
1. Run the chatbot using `python chatbot.py`
2. Interact with the chatbot in the console.

## Features
- Swear word protection
- Custom responses for specific phrases
- Integration with Groq API for generating responses.
